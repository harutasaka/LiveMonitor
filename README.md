# LiveMonitor

> 一款轻量级的跨平台直播主播开播监控桌面插件，支持系统托盘后台运行。
> (release下载LiveMonitor.exe直接开始使用)

---

## 应用简介

**LiveMonitor** 是一款专为直播观众设计的桌面开播提醒工具。它支持 Bilibili、斗鱼、虎牙，能够自动获取主播头像与名称，实时监控开播状态，并以最小化系统托盘的方式常驻后台，开播时通过状态圆点即时反馈，帮助用户不错过任何一场直播。

---

## 功能介绍

### 核心监控
- **多平台支持**：Bilibili、斗鱼（Douyu）、虎牙（Huya）
- **自动信息获取**：输入房间号即可自动拉取主播真实昵称与头像（支持 Bilibili / 斗鱼 / 虎牙）
- **实时开播状态**：头像右下角绿色圆点表示直播中，红色圆点表示未开播
- **智能排序**：开播主播自动置顶，同状态下按字母排序
- **自动刷新**：默认每 60 秒自动检测一次全部主播状态

### 交互操作
- **一键进入直播间**：点击主播头像或名称即可跳转对应直播间
- **搜索过滤**：顶部搜索栏实时过滤主播名称
- **添加主播**：通过平台 + 房间号快速添加，名称留空自动获取
- **删除主播**：鼠标悬停显示红色删除按钮，支持二次确认
- **置顶切换**：标题栏 📌 按钮切换「窗口置顶」与「仅桌面显示」

### 系统级功能
- **系统托盘常驻**：点击右上角 × 关闭后最小化到系统托盘，后台持续监控
- **托盘菜单**：右键托盘图标可显示 / 隐藏窗口、彻底退出程序
- **开机自启动**：设置面板一键开启 Windows 开机自动运行
- **多语言切换**：支持 English / 中文 界面语言实时切换

### 界面风格
- **黑色毛玻璃主题**：半透明深色背景，适配各类桌面壁纸
- **无边框窗口**：自定义标题栏，支持鼠标拖拽移动
- **极简布局**：开播状态以色块直观区分，信息一目了然

## 注意事项

- **斗鱼 API**：使用 `http://` 协议，确保网络环境允许非加密请求
- **虎牙 API**：依赖公开缓存接口，若平台调整可能导致短暂失效
- **Windows 图标缓存**：修改图标后若未生效，执行 `ie4uinit.exe -show` 刷新
